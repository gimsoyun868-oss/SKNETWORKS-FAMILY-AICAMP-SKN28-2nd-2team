# SKNETWORKS-FAMILY-AICAMP-SKN28-2nd-2team

# Netflix Customer Churn Prediction

## 1. 프로젝트 개요

### 프로젝트명

Netflix 가입 고객 이탈 예측 프로젝트

### 프로젝트 소개

본 프로젝트는 Netflix 고객 데이터를 기반으로 사용자 행동 패턴과 구독 특성을 분석하여
고객의 이탈 여부를 사전에 예측하는 머신러닝 및 딥러닝 기반 분류 모델을 구축하는 것을 목표로 한다.

단순한 이탈 여부 예측을 넘어, 이탈에 영향을 미치는 주요 요인을 분석하고
이를 기반으로 고객 유지 전략 수립에 활용 가능한 인사이트를 도출하는 데 목적이 있다.

---

## 2. 프로젝트 목표

* 비즈니스 문제 정의 및 데이터 기반 해결 전략 수립
* 고객 행동 데이터 기반 탐색적 데이터 분석(EDA) 수행
* 머신러닝 및 딥러닝 모델 학습 및 성능 비교
* 모델 해석을 통한 주요 이탈 요인 도출
* Streamlit 기반 예측 서비스 구현

---

## 3. 문제 정의 (Business Understanding)

OTT 서비스 산업에서는 신규 고객 확보 비용이 기존 고객 유지 비용보다 높은 경우가 많기 때문에
고객 이탈을 사전에 예측하고 대응하는 것이 중요한 과제로 작용한다.

본 프로젝트는 고객의 연령, 시청 시간, 최근 접속일, 구독 유형, 결제 방식, 선호 장르 등의 정보를 활용하여
고객의 이탈 여부(`churned`)를 예측하는 모델을 구축한다.

이를 통해 이탈 가능성이 높은 고객을 사전에 식별하고,
고객 특성에 기반한 유지 전략을 수립하는 데 기여할 수 있다.

---

## 4. 기대 효과

* 이탈 가능성이 높은 고객 조기 식별
* 고객 세그먼트 기반 맞춤형 유지 전략 수립
* 요금제 및 결제 방식에 대한 개선 인사이트 도출
* 데이터 기반 의사결정 지원

---

## 5. 데이터 설명

### 데이터셋 개요

* 파일명: netflix_customer_churn.csv
* 데이터 수: 5,000건
* 변수 수: 14개
* 결측치: 없음

---

### 주요 변수 설명

| 변수명                    | 설명               |
| ---------------------- | ---------------- |
| customer_id            | 고객 고유 식별자        |
| age                    | 고객 연령            |
| gender                 | 성별               |
| subscription_type      | 구독 유형            |
| watch_hours            | 총 시청 시간          |
| last_login_days        | 마지막 로그인 이후 경과 일수 |
| region                 | 지역               |
| device                 | 이용 디바이스          |
| monthly_fee            | 월 구독 요금          |
| payment_method         | 결제 방식            |
| number_of_profiles     | 프로필 수            |
| avg_watch_time_per_day | 일 평균 시청 시간       |
| favorite_genre         | 선호 장르            |
| churned                | 이탈 여부            |

---

### 타겟 변수 정의

* churned = 1 : 이탈 고객
* churned = 0 : 유지 고객

---

## 6. 분석 접근 방법

본 프로젝트는 다음과 같은 데이터 분석 절차를 따른다.

1. 탐색적 데이터 분석 (EDA)

   * 데이터 분포 및 이상치 확인
   * 변수 간 관계 분석
   * 이탈 여부와의 상관성 분석

2. 데이터 전처리

   * 범주형 변수 인코딩
   * 수치형 변수 스케일링
   * 학습 및 테스트 데이터 분리

3. 모델링

   * Logistic Regression
   * Random Forest
   * Gradient Boosting 계열 모델
   * Deep Neural Network

4. 성능 평가

   * Accuracy
   * Precision
   * Recall
   * F1-score
   * ROC-AUC
   * Confusion Matrix
